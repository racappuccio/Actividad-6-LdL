# Sistema de Estadísticas de Fútbol

Sistema para gestionar estadísticas de jugadores de fútbol.

## Funcionalidades
- Agregar jugadores (Arquero, Defensor, Central, Delantero)
- Mostrar todos los jugadores
- Buscar por apellido
- Editar y borrar jugadores
